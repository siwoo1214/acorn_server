


웹서버란?



웹페이지를 제공해 주는 서버 (컴퓨터, 또는 프로그램)이다 
웹페이지는 정적웹페이지와 동적 웹페이지로 나뉜다 

서버가 제공하는거 두가지
정적웹페이지 - 미리 작성된 html,css, javascript (모든사람들이 똑같은 웹페이지를 받는 것) 톰캣으로 올리면 끝임 이거는
동적웹페이지 - 요청 될 때 만들어서 제공하는 웹페이지 (백엔드를 만드는 것은 동적인 웹페이지를 만든다는 것을 말한다)
서블릿,JSP,서블릿/JSP로 다 만들 수 있음
마지막 프로젝트는 Spring Boot로 만든다

자바가 동적인 기능을 갖게 해주는 거 - Servlet(클래스)

서블릿 (프로그램) - 이 프로그램이 실행되면서 실행된 결과를 웹페이지 형식으로 제공한다   => 동적 웹페이지라고 한다 !!
(서블릿이라는 프로그램의 실행결과가 응답되는 것이다. html형식의 데이터가 응답됨)
